# Check Console

Read browser console for errors and warnings.

## Instructions

1. Use browser tools to read console messages

2. Filter for:
   - Errors (red)
   - Warnings (yellow)
   - Failed network requests

3. Ignore:
   - Info/debug messages
   - Third-party library noise (unless causing issues)

4. For each error/warning found:
   - Show the message
   - Identify the source file if available
   - Suggest a fix

5. If no errors: confirm console is clean
